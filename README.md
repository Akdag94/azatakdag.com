# azatakdag.com

Personal portfolio website — live at [azatakdag.com](https://azatakdag.com).

Cinematic single-page experience with GSAP-driven scroll animations, smooth scrolling (Lenis), dark/light themes and TR/EN localization.

## Tech stack

- **Framework:** Next.js (App Router) · React 19 · TypeScript
- **Styling:** Tailwind CSS v4
- **Animation:** GSAP + Lenis smooth scroll
- **UI:** Radix primitives, lucide icons

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm start
```
